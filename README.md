# Mobile-Sales-Analysis-

This project analyzes mobile sales data to understand overall performance, profit trends, and brand growth.
Using Power BI, I created an interactive dashboard that shows key insights like top-selling brands, regional sales comparison, and monthly growth trends.

The goal of this project is to make sales data easy to understand and help identify areas for business improvement.

Tools used: Power BI, Excel
